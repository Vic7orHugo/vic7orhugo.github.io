<SOBRE />

Sou graduado em Engenharia Eletrônica pela Universidade Federal de Sergipe, Brasil. Meu Trabalho de Conclusão de Curso foi sobre Controle Digital, ou para ser mais específico, Estimação e Controle no Espaço de Estados. Durante eu curso, eu adquiri gosto por Reconhecimento de Padrões e descobri que minha preferência é na área de programação.

Eu gostei de programar desde o momento que eu comecei a aprender durante meu curso técnico de eletrônica. Estou começando a desenvolver pequenos projetos web, e aprendendo todo o conteúdo necessário para me tornar um desenvolvedor web com foco maior no lado do servidor (back-end), mas com capacidade de trabalhar com projetos tanto para o lado do cliente (front-end) quanto ambos simultaneamente (full-stack). Até este momento, eu obti conhecimentos em HTML5, CSS3, Bootstrap4, Sass(Scss), JavaScript, jQuery, React.js, Node.js, Express.js e MongoDB através de cursos dos sites FreeCodecamp, EDX and Codecademy.

Jogo do Simon®

Esta foi uma implementação do classico Jogo do Simon, onde um jogador será apresentado com uma sequência de botões para pressionar na ordem certa. O usuário pode escolher entre o modo que permite erros (strict mode desligado) ou não (strict mode ligado). Esse foi um projeto feito para o curso do FreeCodecamp.

Jogo da Velha

Uma simples página web para jogar o jogo da velha, onde o usuário terá as opções de escolher para jogar usando "O" ou "X", e escolher jogar contra outra pessoa ou o computador. A inteligência artificial do computador usa o conhecido algoritmo Minimax. Este projeto também foi implementado para o curso dp FreeCodecamp.

Relógio Pomodoro

Esta é uma página web que executa um relógio Pomodoro, um cronômetro para o uso do cliente que utiliza a Técnica Pomodoro. O usuário pode configurar os cronômetros para sessão de trabalho e repouso em intervalos de tempo entre 1 e 90 minutos. Este foi outro projeto realizado para o curso do FreeCodecamp.
