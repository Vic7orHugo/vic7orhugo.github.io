<PORTFOLIO />

Simon® Game

This was an implementation of the classic Simon Game, where the player will be presented with a sequence of buttons to press in the right order, and the user can choose to be able to make mistakes (strict mode OFF) or not (strict mode ON). This was a task from FreeCodecamp.

Tic Tac Toe

A simple webpage to play the game Tic Tac Toe that gives the user the choice to play with "O" or "X", and against another player or the computer. The AI uses the already known Minimax algorithm. This was also a task from FreeCodecamp.

Pomodoro Clock

This is a webpage that executes a Pomodoro Clock, a timer for the client to use the Pomodoro Technique. The user can configure the timer for session and break intervals between 1 and 90 minutes. This was another task from FreeCodecamp.
